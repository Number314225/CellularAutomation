玩法
----------------------

在canvas上点击切换细胞的生死状态

右下角控制按钮 run运行 stop暂停 clear清屏 至于open... 暂时是个坑

也可以在run的状态下切换细胞生死状态

更具体的规则 wiki link https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life
